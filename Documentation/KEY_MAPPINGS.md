## Layout Overview

Rabiraag keyboard utilizes **four layers**:

1. **Normal** - Default keypresses (no modifiers)
2. **Shift** - Holding the Shift key
3. **AltGr** - Holding Right Alt (or Ctrl + Alt)
4. **AltGr + Shift** - Holding Right Alt + Shift

**Special behaviors:**
* **CapsLock:** Works independently. Typically, MSKLC layouts handle Shift states distinctly from CapsLock, but typing remains fluid.
* **NumPad:** Always types Western numerals (0-9).
* **Empty layers:** Undefined combinations type nothing to prevent accidental keystrokes.

## Modifier Key Behavior

### AltGr (Right Alt) in Windows
On Windows keyboards, the **Right Alt** key functions directly as `AltGr`. 
If your keyboard lacks a physical Right Alt key, you can consistently trigger the AltGr layer by pressing **`Ctrl + Alt`** simultaneously.

### Left Alt & Windows System Shortcuts
Because this is a native MSKLC layout, it respects all standard Windows shortcuts:
* `Alt + Tab` → Switch windows
* `Ctrl + C` / `Ctrl + V` → Copy / Paste
* `Win + Space` → Switch between Rabiraag and your English keyboard
* `Win + D` → Show desktop

All system-level shortcuts will function exactly as expected without interfering with your Bengali typing.
