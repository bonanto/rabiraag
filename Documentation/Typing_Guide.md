\# টাইপিং গাইড / Typing Guide (Rabiraag Keyboard)



রবিরাগ কীবোর্ডে আপনাকে স্বাগত! এটি একটি ইনস্ক্রিপ্ট (Inscript) কীবোর্ড লেআউট, যার অর্থ হলো— এখানে কোনো ফোনেটিক বা অনুমানের বিষয় নেই। আপনি ঠিক যেমনটা চাইবেন, কীবোর্ড ঠিক তেমনটাই লিখবে। আপনার বানান এবং যুক্তাক্ষরের ওপর আপনার সম্পূর্ণ নিয়ন্ত্রণ থাকবে।



Welcome to the Rabiraag Keyboard! This is an Inscript layout, meaning there is no phonetic guesswork involved. You have absolute, 100% control over every character, conjunct, and symbol you type.

\## 🇮🇳 বাংলা টাইপিং গাইড (Bengali Guide)



রবিরাগ কীবোর্ডে টাইপ করা অত্যন্ত সহজ এবং যুক্তিসঙ্গত। ফোনেটিক কীবোর্ডের মতো ইংরেজি বানানের ওপর নির্ভর না করে, আপনি সরাসরি বাংলা বর্ণমালার নিয়ম অনুযায়ী টাইপ করবেন।



\### ১. সাধারণ অক্ষর এবং কার (Basic Characters \& Vowels)

স্বরবর্ণের কার (া, ি, ু, ে, ো ইত্যাদি) লেখার জন্য, প্রথমে ব্যঞ্জনবর্ণটি টাইপ করুন এবং তারপর কার-এর বোতামটি চাপুন।

\* \*\*মা (Ma):\*\* `m` (ম) + `a` (া) = মা

\* \*\*দিন (Din):\*\* `v` (দ) + `i` (ি) + `n` (ন) = দিন

\* \*\*শুরু (Shuru):\*\* `Shift+S` (শ) + `u` (ু) + `r` (র) + `u` (ু) = শুরু



\### ২. যুক্তাক্ষর ও জটিল শব্দ তৈরি করা (Making Conjuncts \& Complex Words)

যুক্তাক্ষর তৈরি করতে আমরা \*\*হসন্ত (্)\*\* ব্যবহার করি। কীবোর্ডে হসন্ত রয়েছে \*\*`\*\* (Backtick/Tilde) বোতামটিতে (এটি Tab বোতামের ঠিক ওপরে থাকে)।

যেকোনো দুটি বা তিনটি ব্যঞ্জনবর্ণকে যুক্ত করতে তাদের মাঝে কেবল একবার হসন্ত ( ` ) বোতামটি চাপুন।



\*\*নিয়ম:\*\* প্রথম অক্ষর + হসন্ত ( ` ) + দ্বিতীয় অক্ষর = যুক্তাক্ষর



\*(রেফ লেখার নিয়ম: যে অক্ষরের মাথায় রেফ বসবে, তার আগে 'র + ্' টাইপ করতে হয়। যেমন: র + ্ + ত = র্ত)\*



\*\*কয়েকটি সাধারণ ও জটিল শব্দের উদাহরণ:\*\*



| শব্দ (Word) | গঠন (Structure) | কীবোর্ডে যা টাইপ করবেন (Keystrokes) |

| :--- | :--- | :--- |

| \*\*ক্ত\*\* | ক + ্ + ত | `k` + `` ` `` + `x` |

| \*\*বনান্ত\*\* | ব + ন + া + ন + ্ + ত | `b` `n` `a` `n` `` ` `` `x` |

| \*\*শাশ্বতী\*\* | শ + া + শ + ্ + ব + ত + ী | `Shift+S` `a` `Shift+S` `` ` `` `b` `x` `Shift+I` |

\### ৩. সরাসরি যুক্তাক্ষর (Pre-assigned Conjuncts)

কিছু বহুল ব্যবহৃত যুক্তাক্ষর সরাসরি লেয়ার ৩ (AltGr) বা লেয়ার ৪ (AltGr+Shift) এ দেওয়া আছে। যেমন:

\* \*\*ক্ষ\*\* = `q`

\* \*\*জ্ঞ\*\* = `AltGr+g`

আপনি চাইলে "জ + ্ + ঞ" (`j` + `` ` `` + `Shift+W`) টাইপ করেও 'জ্ঞ' লিখতে পারেন, আবার সরাসরি `AltGr+g` ও ব্যবহার করতে পারেন।



\### ৪. ZWNJ (Zero Width Non-Joiner) এর ব্যবহার

কখনো কখনো আমরা চাই হসন্ত থাকুক, কিন্তু অক্ষরগুলো যেন যুক্ত না হয়। এর জন্য \*\*ZWNJ\*\* ব্যবহার করা হয়। রবিরাগে ZWNJ রয়েছে \*\*Shift + `\*\* (Shift এবং হসন্ত বোতামটি একত্রে) এ।



\* \*\*ক্‌ত (যুক্ত না করে):\*\* ক + ্ + ZWNJ + ত 

&nbsp; \* \*\*টাইপ করুন:\*\* `k` + `` ` `` + `Shift+` `` `x`

\* \*\*র‌্যাম (RAM):\*\* র + ZWNJ + ্ + য + া + ম

&nbsp; \* \*\*টাইপ করুন:\*\* `r` + `Shift+` `` `` ` `` + `y` + `a` + `m`



প্রথমদিকে দেখে হয়তো একটু কঠিন মনে হতে পারে, কিন্তু একবার চেষ্টা করে দেখুন! এটি অত্যন্ত সহজ, এবং খুব অল্প সময়ের মধ্যেই আপনি সম্পূর্ণ নিয়ন্ত্রণ নিয়ে দ্রুতগতিতে নির্ভুল বাংলা টাইপ করা শিখে যাবেন।

\## 🇬🇧 English Typing Guide



Typing on the Rabiraag keyboard follows the logical structure of the Bengali script. Instead of relying on software to guess the word from English phonetic spellings, you build the word exactly as it is written.



\### 1. Basic Characters and Vowel Modifiers (Kar)

To type a consonant with a vowel modifier (like া, ি, ু, etc.), simply type the consonant first, followed by the modifier key.

\* \*\*মা (Ma):\*\* `m` (ম) + `a` (া) = মা

\* \*\*দিন (Din):\*\* `v` (দ) + `i` (ি) + `n` (ন) = দিন

\* \*\*শুরু (Shuru):\*\* `Shift+S` (শ) + `u` (ু) + `r` (র) + `u` (ু) = শুরু



\### 2. Creating Conjuncts \& Complex Words (The Hasanta Key)

To join two or more consonants together (যুক্তাক্ষর), you must use the \*\*Hasanta/Virama (্)\*\*.

The Hasanta is located on the \*\*`\*\* (Backtick/Tilde) key, which is situated just above the `Tab` key on your keyboard.



\*\*Formula:\*\* First Character + Hasanta ( ` ) + Second Character = Conjunct



\*(Note on Reph: To add a Reph over a consonant, you type 'র + ্' right before the consonant. Example: র + ্ + ত = র্ত)\*



\*\*Step-by-Step Examples of Simple and Complex Words:\*\*



| Word | Logic | What you type on the Keyboard |

| :--- | :--- | :--- |

| \*\*ক্ত\*\* | ক + ্ + ত | `k` + `` ` `` + `x` |

| \*\*বনান্ত (Bonanto)\*\* | ব + ন + া + ন + ্ + ত | `b` `n` `a` `n` `` ` `` `x` |

| \*\*শাশ্বতী (Saswati)\*\* | শ + া + শ + ্ + ব + ত + ী | `Shift+S` `a` `Shift+S` `` ` `` `b` `x` `Shift+I` |

| \*\*শ্যামল (Shyamal)\*\* | শ + ্ + য + া + ম + ল | `Shift+S` `` ` `` `y` `a` `m` `l` |

| \*\*প্রকৃতি (Prakriti)\*\* | প + ্ + র + ক + ৃ + ত + ি | `p` `` ` `` `r` `k` `z` `x` `i` |

| \*\*উজ্জ্বল (Ujjwal)\*\* | উ + জ + ্ + জ + ্ + ব + ল | `AltGr+u` `j` `` ` `` `j` `` ` `` `b` `l` |

\*(Note: You do not need to type the plus (+) signs. Just press the keys sequentially. Vowel modifiers are always typed after the consonant.)\*



\### 3. Direct Conjuncts (Shortcuts)

To save time, some highly common conjuncts are assigned directly to the Normal layer or AltGr layer.

\* \*\*ক্ষ\*\* = simply press `q`

\* \*\*জ্ঞ\*\* = press `AltGr+g`

You can always manually construct them (e.g., `j` + `` ` `` + `Shift+W` for জ্ঞ), but the shortcuts are there for your convenience.



\### 4. Preventing Conjuncts (Using ZWNJ)

Sometimes you need to use the Hasanta (্) but specifically \*\*do not\*\* want the letters to combine into a conjunct. This requires a \*\*Zero Width Non-Joiner (ZWNJ)\*\*.

In Rabiraag, the ZWNJ is mapped to \*\*Shift + `\*\* (Shift + the Hasanta key).



\* \*\*Example 1: ক্‌ত (Visible Hasanta, no joining)\*\*

&nbsp; \* \*\*Logic:\*\* ক + ্ + ZWNJ + ত

&nbsp; \* \*\*Type:\*\* `k` + `` ` `` + `Shift+` `` `x`

\* \*\*Example 2: র‌্যাম (RAM)\*\*

&nbsp; \* \*\*Logic:\*\* র + ZWNJ + ্ + য + া + ম

&nbsp; \* \*\*Type:\*\* `r` + `Shift+` `` `` ` `` + `y` + `a` + `m`



These instructions might look a bit intimidating at first glance, but try it out! It is extremely simple. Once your fingers memorize the locations, you will learn to type Bangla in no time, with absolute and unparalleled control over the characters, signs, and conjuncts.
